# 📄 Generador de CV Inteligente

[![Ask DeepWiki](https://deepwiki.com/badge.svg)](https://deepwiki.com/Gajeta21/CV-Generator)

Generador de currículums vitae con interfaz paso a paso, múltiples plantillas profesionales, auto‑guardado, importación/exportación de datos y optimización para sistemas de seguimiento de candidatos (ATS). Diseñado para que cualquier persona pueda crear un CV atractivo y compatible con los sistemas de selección automatizados.

> 🔗 **Repositorio:** [Gajeta21/CV-Generator](https://github.com/Gajeta21/CV-Generator)  
> 🖥️ **Demo en vivo:** [Enlace a GitHub Pages](https://gajeta21.github.io/CV-Generator/)

---

## ✨ Características principales

- **Interfaz por pasos** (0-9): Datos personales, experiencia, educación, referencias, logros, proyectos, habilidades, idiomas, certificaciones y vista previa.
- **6 plantillas de CV** + una **plantilla ATS Pro** optimizada para superar filtros automáticos.
- **Guardado automático** en el navegador (localStorage) — nunca pierdas tu progreso.
- **Importar/exportar** todos los datos a un archivo JSON (útil para respaldos o cambiar de dispositivo).
- **Botón de impresión** que genera un PDF listo para enviar o imprimir.
- **Validación en tiempo real** de teléfonos, fechas y campos obligatorios.
- **Indicador de completitud** que te guía para llenar todas las secciones importantes.
- **Compatible con ATS** (formato de una columna, encabezados semánticos, palabras clave).

---

## 🧩 Secciones incluidas en el CV

- **Datos personales**: nombre, fecha nacimiento, dirección, teléfonos, email, redes sociales, foto de perfil.
- **Resumen profesional** (opcional).
- **Experiencia laboral** (múltiples entradas con fechas y descripciones).
- **Educación** (estudios, instituciones, fechas, estado, duración).
- **Referencias** (nombre, teléfono, relación).
- **Logros destacados** (lista de texto).
- **Proyectos personales** (nombre, descripción, tecnologías, enlace).
- **Habilidades** (lista de palabras clave).
- **Idiomas** (nombre y nivel según MCER).
- **Certificaciones** (nombre, institución, fecha, enlace).
